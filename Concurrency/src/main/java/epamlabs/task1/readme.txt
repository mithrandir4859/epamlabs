Написать программу, подсчитывающую в M потоках сумму ряда sin(x)*cos(x),
где x пробегает диапазон от -N до N, с шагом step = 0.0001
sum = sin(-N)*cos(-N) + …  + sin(N)*cos(N)
При реализации не использовать блоки синхронизации и библиотеку java.util.concurrent
Провести тестирование и определить, в скольких потоках оптимальное считать сумму ряда
(зависимость времени подсчета от числа потоков).
